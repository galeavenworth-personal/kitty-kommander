# Sprite System

Visual status language for the cockpit DAG. Beads render as yarn balls. Kitties
show ownership and state.

## Yarn Ball Nodes (`nodes/`)

Each bead state has a yarn ball sprite. Generated by `generate.py` (requires
Pillow), used by `cockpit_dash.py` as Graphviz image nodes.

| Sprite | State | Visual |
|--------|-------|--------|
| `yarn_ready.png` | Ready | Green (#9ece6a), tight and clean |
| `yarn_blocked.png` | Blocked | Red (#f7768e), tense |
| `yarn_wip.png` | In progress | Yellow (#e0af68), active |
| `yarn_open.png` | Open | Grey (#565f89), dim, waiting |
| `yarn_done.png` | Done | Blue (#7aa2f7), settled |

Size: 64x64 PNG, transparent background. Rendered at 256x256 and downscaled.

## Kitty Sprites (`kitties/`) — planned

State-based sprite system. Each visible kitty answers one question:
**who is working, what are they doing, or what just changed.**

### Roles (6)

| Role | Accent | Filename prefix | Character |
|------|--------|-----------------|-----------|
| Kommander | White/silver (#a9b1d6) | `kommander_` | Regal posture, commands the cockpit. The conductor. |
| Lead | Gold/amber (#e0af68) | `lead_` | Seated upright, one paw on console. Oversees the yarn web. |
| Builder | Orange (#ff9e64) | `builder_` | Typing, ears forward, tail tucked. Making the thing. |
| Scout | Cyan (#7dcfff) | `scout_` | Leaning forward, peering. Inspecting. |
| Critic | Violet (#bb9af7) | `critic_` | Paw up, narrowed eyes. Reviewing. |
| Integrator | Green (#9ece6a) | `integrator_` | Carrying or arranging yarn balls. Merging. |

Color is the primary role discriminator. At badge size (16x16) the
silhouette may be too small for posture to read — accent color must
carry the identity alone.

### States (7, universal across all roles)

| State | Pose | Filename suffix | When |
|-------|------|-----------------|----- |
| idle | Sitting, tail wrapped | `_idle` | Waiting for work |
| active | Typing, paws on controls | `_active` | Executing a task |
| thinking | Head tilted, paw raised | `_thinking` | Planning or analyzing |
| blocked | Caution indicator | `_blocked` | Stalled on a dependency |
| handoff | Carrying yarn ball | `_handoff` | Passing work to another agent |
| done | Tail up, settled | `_done` | Task complete |
| alert | Ears up, straight posture | `_alert` | Escalation or attention needed |

### File naming convention

```
sprites/kitties/{size}/{role}_{state}.png
```

Examples:
- `sprites/kitties/panel/kommander_active.png` (32x32)
- `sprites/kitties/badge/builder_blocked.png` (16x16)
- `sprites/kitties/focus/scout_thinking.png` (48x48)

The sprite picker is a pure function:
```python
def kitty_sprite(role, state, size="panel"):
    return SPRITE_DIR / "kitties" / size / f"{role}_{state}.png"
```

### Matrix summary

6 roles x 7 states = **42 sprites per size tier**.

Priority tiers for production:
1. **Panel (32x32)**: All 42. This is the primary working size.
2. **Badge (16x16)**: All 42. DAG node overlays need these.
3. **Focus (48x48)**: Common states only (idle, active, blocked, done)
   = 24 sprites. Detail view, selected agent.
4. **Compact (24x24)**: Stretch goal. Task lists, agent rosters.

### Sizes

| Tier | Size | Use |
|------|------|-----|
| Badge | 16x16 | DAG node overlays, edge events |
| Compact | 24x24 | Task board rows, agent lists |
| Panel | 32x32 | Side panels, team roster |
| Focus | 48x48 | Detail view, selected agent |

### DAG Placement

- **Node owner sprite**: Tiny role-colored kitty badge anchored to the yarn ball.
- **Node state sprite**: Kitty pose changes with node state.
- **Edge activity sprite**: Temporary moving kitty during handoff or propagation.

### Action Sprites (transient)

| Action | Visual |
|--------|--------|
| Walking along edge | Work moving downstream |
| Carrying yarn ball | Handoff occurred |
| Perched on node | Node currently owned |
| Sleeping on node | Idle or waiting |
| Waving flag | Blocker or escalation |
| Sparkle | Completion |

### Discipline

Persistent kitties show ownership and state.
Transient kitties show change.
Decorative kitties should be rare.

If a kitty doesn't answer "who, what, or what changed" — it shouldn't be there.
